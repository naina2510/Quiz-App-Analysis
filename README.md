# Quiz App with Performance Analysis

## 📌 Project Overview
This project is a Python-based Quiz Application developed using Tkinter/CustomTkinter. It allows users to take quizzes, stores results, and provides performance analysis through an admin dashboard.

## 🚀 Features
- User Login
- Random Quiz Questions
- Score Calculation
- Performance Analysis
- Admin Dashboard
- Excel Data Storage
- Graphical User Interface (GUI)

## 🛠️ Technologies Used
- Python
- Tkinter / CustomTkinter
- Pandas
- Matplotlib
- OpenPyXL

## 📂 Project Structure

```
quiz_gui_final.py
admin_credentials.xlsx
quiz_results_flat.xlsx
quiz_results.csv
quiz_log.csv
README.md
requirements.txt
.gitignore
```

## ▶️ How to Run

```bash
pip install -r requirements.txt
python quiz_gui_final.py
```
